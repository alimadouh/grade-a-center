# Freelance Project Tracker (Vite + React + Tailwind)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Netlify

- Build command: `npm run build`
- Publish directory: `dist`

Notes:
- Data is stored in your browser using localStorage.
- "Download Receipt History" opens a print dialog. Choose "Save as PDF".
